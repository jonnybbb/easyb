Downloading easyb

easyb is distributed as a archive containing an easyb jar file along with its associated dependencies (including Groovy). easyb plug-ins (like DbUnit) are now packaged separately-- you can download them with an associated release here as well.

The current version of easyb is 0.6

  • easyb-0.6.tar.gz (January 2008)
    • easyb-dbunit-plugin-0.6.tar.gz (January 2008)

Older public releases:

  • easyb-0.02.tar.gz (November 2007)

If you have any questions or comments, drop us a note on our mailing list.
