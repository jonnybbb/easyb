Downloading easyb

easyb is distributed as a archive containing an easyb jar file along with its associated dependencies (including Groovy). easyb plug-ins (like DbUnit) are packaged separately-- you can download them with an associated release here as well.

The current version of easyb is 0.9.5 (March 2009) and can be downloaded from easyb's project site at Google code. Note, there are different archives for downloading-- easyb itself, the dbunit plugin, and an IntelliJ plugin.

Older public releases:

  • easyb-0.9.tar.gz (July 2008)
    • easyb-dbunit-plugin-0.9.tar.gz (July 2008)
    • easyb-0.8.tar.gz (April 2008)
      • easyb-dbunit-plugin-0.8.tar.gz (April 2008)
    • easyb-0.7.tar.gz (February 2008)
      • easyb-dbunit-plugin-0.7.tar.gz (February 2008)
    • easyb-0.6.tar.gz (January 2008)
      • easyb-dbunit-plugin-0.6.tar.gz (January 2008)
    • easyb-0.02.tar.gz (November 2007)
